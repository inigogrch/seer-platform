# Seer Retrieval Agent

AI-powered news aggregation system for tech professionals.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Copy `.env.example` to `.env` and add your API keys:
```bash
cp .env.example .env
```

3. Run tests:
```bash
pytest
```
